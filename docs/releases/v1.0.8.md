# v1.0.8 — npm package rename and release automation

Public packaging release after v1.0.7.

## Changes

- Renamed the npm package to `openclaw-claude-proxy`.
- Kept the executable command as `claude-proxy`.
- Added GitHub Actions release packaging so version tags automatically build and upload `.tgz` and `.sha256` assets.
- Published `openclaw-claude-proxy@1.0.8` to npm and kept the GitHub release tarball as an alternative.

## Install

```bash
npm install -g openclaw-claude-proxy
claude-proxy
```

GitHub release tarball alternative:

```bash
npm install -g https://github.com/mehdic/openclaw-claude-proxy/releases/download/v1.0.8/openclaw-claude-proxy-1.0.8.tgz
claude-proxy
```

## Validation

- `npm run build`
- `npm test`
- `npm pack`
- fresh global install smoke from the generated tarball
- npm registry publish verified with `npm view openclaw-claude-proxy version`
