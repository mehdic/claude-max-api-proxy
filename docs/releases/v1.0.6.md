# claude-proxy v1.0.6

Operational follow-up to v1.0.5.

## Highlights

- Installed and pinned optional JS SDK matrix dependencies: OpenAI Node and LangChain JS.
- Added `SDK_MATRIX_PYTHON` support plus `npm run setup:sdk-matrix` for a local Python OpenAI SDK venv.
- Added durable SQLite trace retention pruning via `CLAUDE_PROXY_TRACE_SQLITE_RETENTION_DAYS` / `CLAUDE_PROXY_TRACE_SQLITE_RETENTION_MS`.
- Enabled live SQLite traces with 7-day retention on the production LaunchAgent.
- Added `npm run monitor:live`, a lightweight `/health` + tiny chat smoke monitor.
- Loaded daily macOS LaunchAgent `ai.openclaw.claude-proxy-monitor` for failure-only operator alerts.
- Added trace security documentation and an OpenClaw integration check document.
- Cleared npm audit vulnerabilities without force-upgrading blindly.

## Validation

- `npm audit --audit-level=moderate` → 0 vulnerabilities
